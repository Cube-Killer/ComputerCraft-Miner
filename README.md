#Computer-Craft
Lua programs in ComputerCraft  

#Notice
Your ComputerCraft mod version must be over v1.64.  
Make sure that your turtles has fuels.  

#Useage
##Miner
How to put items:

| Number of Grid | Item Name |
| :-----------------: | :----------------: |
| 1                   | Torch              |
| 2                   | Cubbestone         |

Make sure that your mining-turtle has a wireless modem.
